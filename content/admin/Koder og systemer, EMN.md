# Fargekoding
## XLR-kablene
- Jording: Pin 1
- Blå: Pin 3
- Hvit: Pin 2
## Hann, sideplate (fra venstre)
			Blå
Jord              Hvit

## Hunn, kabelende
			Blå
Hvit              Jord

## Pedaler
**Cloth AWG 22, solid pushback**
### Farger hos Banzai:
- Svart
- Blå
- Brun
- Rød
- Hvit
- Gul
- (Grønn)
### Effekter
**Rød**: Strøm
**Svart**: Jording
**Gul**: LED
**Hvit**: Input -> Bryter
**Grønn**: Bryter -> PCB
**Brun**: PCB -> Bryter
**Blå**: Bryter -> Output
